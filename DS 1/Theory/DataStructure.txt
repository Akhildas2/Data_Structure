1.What is data structure?

    - A data structure is a way to store and organize data so that it can be used efficiently.
    - A data structure is a collection of data values, the relationships among them, and the functions or operations that can be applied to them.

2.Why we use the data structure?

    - Efficiency:
        Efficiency: By organizing data efficiently, computers can access and manipulate information much faster. 
    - Memory Management:
        Data structures help optimize how memory is used, preventing wasted space and keeping programs running smoothly.
    - Code Reusability:
        Well-defined data structures can be used in different parts of a program or even in other programs, saving development time and making code more consistent.

3.Data structure real world Applications

    - Social Media: They are used to manage your friend lists, news feeds, and search results.
    - Online Shopping: They keep track of items in your shopping cart and recommend products you might like.
    - Navigation Apps: They use graphs (a type of data structure) to map locations and find the fastest route.
    - Gaming: They manage game state, characters, and inventories
